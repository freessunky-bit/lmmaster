# 시작 가이드 (한국어)

> v1 사용자용 가이드 — 실제 사용 단계는 M6 무렵 채워집니다. 현재는 골격만.

## LMmaster란?

기존 웹앱이 호출만 해서 사용할 수 있는 로컬 AI 도우미입니다. 데스크톱에서 실행되며, 모델 설치·런타임 관리·키 발급을 한곳에서 해줍니다.

## 처음 실행하면

1. 환경 점검(내 PC 사양 자동 진단).
2. 추천 모델 3종 표시 (상위 품질 / 균형형 / 경량형).
3. 마음에 드는 모델 선택 → 자동 설치.
4. 워밍업 후 사용 가능 상태로 진입.
