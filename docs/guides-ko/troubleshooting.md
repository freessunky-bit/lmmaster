# 문제 해결 가이드 (한국어)

## 자주 묻는 상황

### 응답이 느려요
- 첫 응답 직전 워밍업 단계인지 확인 (홈 → 상태).
- GPU 가속이 비활성이면 진단 → 환경 점검에서 드라이버 확인.
- 모델이 PC 사양 대비 무거우면 더 작은 양자화로 다운그레이드.

### GPU를 못 잡아요
- 진단 → 환경 점검 → GPU 항목 확인.
- 드라이버 업데이트 가이드 따라 적용 후 LMmaster 재시작.

### 다운로드가 자꾸 끊겨요
- 자동 재시도(resumable)가 동작하는지 설치 센터 확인.
- 디스크 여유 공간 확인.
- 네트워크 안정성/프록시 설정 확인.

### 기존 웹앱이 LMmaster에 연결을 못해요
- LMmaster가 실행 중인지 확인 (시스템 트레이 또는 사이드바 가동 표시).
- 헬스 모달의 [LMmaster 실행] 링크 클릭(`lmmaster://launch`).
- 포트 충돌 시 설정 → 네트워크에서 포트 변경.

## 로그 보내기

진단 → 로그 → "내보내기"로 로그 묶음 생성.
민감 정보는 자동 마스킹 후 zip에 포함됩니다.
