# 모델 설치 가이드 (한국어)

> M2~M4 산출물 — 실제 단계 화면 캡처는 후속 추가.

## 추천부터 시작하기

홈 화면의 추천 카드 3종 중 하나를 고르세요:
- **상위 품질**: 자원이 충분할 때 가장 좋은 결과.
- **균형형**: 품질·속도·자원의 균형.
- **경량형**: 자원 절약 우선.

## 카탈로그에서 직접 고르기

- 좌측 카테고리: 에이전트 / 캐릭터 / 코딩 / 사운드(STT, TTS) / SLM.
- 적합도 라이트(녹/황/적)로 내 PC와의 호환을 확인.
- 양자화 옵션(예: Q4_K_M) 선택 후 설치.

## 설치 진행 단계

다운로드 중 → 검증 중 → 압축 해제 → 워밍업 중 → 사용 가능.
중단되면 [재시도]로 재개. 체크섬 불일치 시 자동으로 재다운로드 옵션 제시.

## 디스크 관리

설정 → 워크스페이스 → 캐시 정책에서 LRU 한도 설정.
사용하지 않는 모델은 카탈로그에서 [삭제] 가능.
