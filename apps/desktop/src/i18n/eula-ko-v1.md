# LMmaster 사용자 동의서 (v1.2.0)

**적용 시점**: 사용자가 본 동의서에 처음 동의한 시점부터 적용돼요. 동의 시점·동의서 버전(v1.2.0)은 사용자 PC 로컬에 저장돼요. 본 동의서가 minor / major 버전으로 갱신되면 다시 동의를 요청해요(§6).

본 동의서는 한국 소비자 보호법령 및 사용자 거주국의 강행규정으로 보장된 사용자 권리를 어떤 식으로도 제한하지 않아요. 본 동의서의 일부 조항이 그 법령과 충돌하면, 그 조항만 효력 범위 안에서 조정되거나 적용되지 않아요. 나머지 조항은 그대로 유효해요.

## 1. 개요

LMmaster는 사용자의 PC에서 로컬로 동작하는 AI 운영 허브예요. 본 동의서는 LMmaster 사용에 관한 사용자 권리·의무를 정의해요.

본 동의서는 LMmaster 본체에만 적용돼요. LM Studio, Ollama, 모델 가중치, Python 의존성 같은 외부 구성 요소는 각 벤더의 라이선스·EULA가 별도로 적용돼요(§8).

## 2. 사용 권한

- 개인적·상업적 용도 모두 사용 가능해요.
- 코드 수정·재배포는 LICENSE 파일의 조건 아래 가능해요.
- 본 앱을 활용해 만든 결과물은 사용자 본인 소유예요.

다만 다음 용도로는 사용을 금지해요. 위반 시 사용자 본인이 모든 법적·도의적 책임을 부담해요.

- 본인 또는 제3자의 법령상 권리(저작권·상표권·초상권·명예권·프라이버시·영업비밀 등)를 침해하는 콘텐츠의 생성·배포.
- 미성년자에 대한 성적·폭력적 묘사 콘텐츠의 생성·소지·배포 (한국 「청소년보호법」, 「아동·청소년의 성보호에 관한 법률」, 미국 PROTECT Act, EU CSAM Regulation 등 위반).
- 보이스피싱·사기·해킹·악성코드·스팸 등 형사 범죄 또는 부정경쟁 목적의 사용.
- 특정 개인·집단에 대한 차별·혐오·위협·증오 표현의 조장 또는 자동 생성.
- 의료·법률·금융 등 면허·자격이 필요한 영역에서 본 앱 출력만으로 전문 자문을 대체하는 행위.

## 3. 외부 통신

- LMmaster 본체는 외부 통신 0 원칙(ADR-0013)을 따라요.
- 자동 업데이트 확인은 GitHub Releases (api.github.com) 1회/6시간 — 사용자가 Settings에서 비활성 가능해요.
- LM Studio · Ollama 같은 외부 런타임은 각 벤더의 EULA가 별도 적용돼요. 우리가 그 EULA를 다시 보여드리지 않아요.
- 텔레메트리 opt-in을 켰을 때만 익명 통계가 단일 endpoint(GlitchTip self-hosted)로 전송돼요.

## 4. 데이터

- 사용자가 입력한 프롬프트·문서·모델은 모두 PC에서만 처리돼요.
- 워크스페이스 데이터는 portable 디렉터리(`%APPDATA%\LMmaster` 등)에 저장돼요.
- 텔레메트리는 기본 비활성. opt-in했을 때만 익명 PC 통계(OS major version / GPU 모델 / VRAM)가 전송돼요. 프롬프트 / 모델 출력 / 파일 내용은 절대 전송되지 않아요.

## 5. 책임 한계

- 본 소프트웨어는 "있는 그대로(AS IS)" 그리고 "이용 가능한 상태대로(AS AVAILABLE)" 제공돼요. 적용 가능한 법령이 허용하는 최대 범위에서, 명시적·묵시적 보증(상품성·특정 목적 적합성·비침해·정확성·중단 없음·결함 없음에 관한 보증을 포함하되 이에 한정되지 않음)을 제공하지 않아요.
- LMmaster 사용 과정에서 발생한 다음 사항에 대해 LMmaster 측은 책임을 지지 않아요.
  - 사용자 PC의 데이터 손실·손상·유실, 워크스페이스 파일 또는 모델 가중치 손상.
  - 모델 출력의 부정확성·편향·환각(hallucination)·불쾌감·법령 위반·제3자 권리 침해.
  - LM Studio·Ollama·llama.cpp·Python 등 외부 런타임의 장애·EULA 위반·라이선스 변경.
  - 외부 카탈로그 fetch 또는 자동 업데이트 확인의 일시적 실패.
- 사용자가 외부에 공개·배포·상업 이용하는 모델 출력의 적법성·윤리성에 대한 책임은 사용자 본인이 부담해요.
- LMmaster는 무상으로 배포되는 소프트웨어이므로, 본 소프트웨어로 인해 사용자가 입은 직접·간접·특별·우발적·결과적·징벌적 손해에 대한 LMmaster 측의 총 책임 한도는 *적용 가능한 법령이 허용하는 최대 범위에서* 사용자가 본 소프트웨어를 받기 위해 LMmaster 측에 실제로 지급한 금액(통상 0원)으로 한정돼요.
- 다만, 위 면책·책임 한도는 LMmaster 측의 고의 또는 중대한 과실로 인한 손해, 또는 한국 「약관의 규제에 관한 법률」·「전자상거래 등에서의 소비자보호에 관한 법률」 및 사용자 거주국의 강행규정에 따라 면책·제한이 허용되지 않는 책임에는 적용되지 않아요. 그 범위에서는 관련 법령에 따라요.

## 6. 변경

본 동의서가 변경되면 사용자에게 알리고 다시 동의를 요청해요.

- patch 버전(예: 1.0.1)은 자동 동의로 갈음해요 (오타 / 명확화).
- minor / major(예: 1.1.0 / 2.0.0)는 재동의가 필요해요 (기능 / 데이터 처리 변경).
- 갱신 시 변경 요약을 함께 보여드려요.

## 7. NSFW 데이터셋 정책 (v0.1.0+)

LMmaster 데이터셋 카탈로그는 다음 정책을 따라요:

1. **미성년자 콘텐츠 금지** — 큐레이터가 자동 키워드 scan(영문/일본어/한국어) + 본문 검토로 1차 거부해요. 사용자가 미성년 묘사 데이터를 발견하면 즉시 신고해 주세요.
2. **HF NFAA 플래그 필수** — Not-For-All-Audiences 플래그가 없는 NSFW 데이터셋은 카탈로그에 등록되지 않아요.
3. **라이선스 화이트리스트** — Apache-2.0 / MIT / OpenRAIL-M / CC-BY 등 검증된 오픈 라이선스만. CC-BY-NC는 *비상업 사용자 동의* 후만 노출돼요.
4. **사용자 책임** — 다운로드 후 사용은 사용자 PC 안에서만 일어나요. 사용자는 본인 국가의 법(한국 청소년보호법 / 미국 PROTECT Act / EU CSAM regulation 등)을 준수할 의무가 있어요.
5. **NSFW 토글** — 카탈로그 헤더의 3-state 토글로 *숨김 / 모두 보임 / 성인만* 사이클 선택 가능해요. 기본은 *숨김*이에요. 모델 NSFW와 데이터셋 NSFW 게이트가 통합돼 있어요.

자세한 정책: `docs/adr/0062-nsfw-dataset-policy.md` 참조.

## 8. 제3자 모델 / 가중치 / 데이터셋 면책 (v1.0.0+)

LMmaster를 통해 사용자가 다운로드·실행하는 모델 가중치(예: Llama / Qwen / Gemma / EXAONE / HCX-SEED 등)와 데이터셋·런타임은 각 제공자의 라이선스·EULA·사용 약관이 별도로 적용돼요.

- LMmaster는 외부 모델·데이터셋·런타임의 출력 정확성·안전성·합법성·라이선스 준수 여부를 보증하지 않아요.
- 모델의 사용·재배포·상업 이용 가능 여부는 사용자가 각 모델 카드와 라이선스 문서를 직접 확인해 주세요. 일부 모델(예: Llama Acceptable Use Policy, Gemma Prohibited Use Policy 등)은 자체 사용 정책을 두고 있어요.
- 모델이 생성한 콘텐츠가 제3자의 저작권·초상권·명예·프라이버시·영업비밀을 침해하지 않는지 확인하고 사용하는 것은 사용자 본인의 책임이에요.
- 모델 가중치·데이터셋 다운로드 과정에서 발생한 네트워크 비용, 디스크 사용량, 보안 위험에 대해서는 LMmaster 측이 책임지지 않아요.

## 9. AI 트렌드 리포트 정책 (v1.2.0+)

LMmaster의 *AI 트렌드 리포트* 메뉴(별도 4B+ 모델 설치 시 활성)는 다음 정책을 따라요:

1. **외부 트렌드 데이터셋 fetch 동의** — 사용자 PC가 `cdn.jsdelivr.net`에서 큐레이션된
   `trends-bundle.json`을 매주 1회 fetch해요. 사용자 PC가 직접 RSS / SNS / 뉴스 사이트를
   scrape하지 않아요 (외부 통신 0 정체성 보존).
2. **큐레이터 흐름** — 별도 repo `lmmaster-trends-bundle`(또는 본 repo prototype)의
   GHA aggregator가 RSS / arXiv / HF Daily Papers / YouTube / Bluesky / Mastodon
   소스를 사람이 검토 후 fair-use 기준 한국어 한 줄 요약으로 변환해 push해요.
3. **로컬 LLM 한국어 요약 정책** — 메뉴 진입 시 사용자 PC의 4B+ 모델(Gemma 3 4B /
   Nemotron 3 Nano 4B / EXAONE 3.5 7.8B / HCX-SEED 8B 등)이 카테고리별로 1~2문장
   해요체 메타 요약을 생성해요. *원문 재출판은 금지*예요. 결과는 30일 캐시돼요.
4. **저작권자 신고 채널** — 본인이 만든 콘텐츠가 부당하게 인용됐다고 판단하시면
   본 repo의 GitHub Issue로 신고해 주세요. 큐레이터가 1주 안에 검토해 다음 push에서
   제외하거나 인용 형식을 조정해요.

자세한 정책: `docs/adr/0060-trend-report.md` + `docs/research/phase-22p-trend-report-decision.md` 참조.

## 11. 일반 조항

- **준거법**: 본 동의서는 대한민국 법률에 따라 해석되고 적용돼요. 다만, 사용자 거주국의 강행규정으로 보장된 사용자 권리는 그대로 유지돼요.
- **분쟁 해결 / 관할법원**: 본 동의서로 인해 발생하는 분쟁은 가급적 LMmaster GitHub Issue 채널에서 우선 협의해요. 협의가 이뤄지지 않으면, 사용자 주소지(또는 거소지)를 관할하는 대한민국 법원을 1심 전속관할 법원으로 해요. 사용자 주소·거소가 명확하지 않을 때는 「민사소송법」상 관할 법원에 따라요.
- **분리가능성**: 본 동의서의 일부 조항이 무효이거나 집행 불가능하다고 판정돼도 나머지 조항의 효력은 영향을 받지 않아요.
- **양도**: 사용자는 본 동의서상의 권리·의무를 LMmaster 측의 사전 동의 없이 제3자에게 양도할 수 없어요.
- **수출 통제**: 사용자는 본 소프트웨어 또는 모델 가중치를 사용자 거주국의 수출 통제 법령(예: 한국 「대외무역법」, 미국 EAR 등)을 위반하는 방식으로 반출·재수출하지 않을 의무가 있어요.
