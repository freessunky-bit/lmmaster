# LMmaster 사용자 동의서

본 동의서는 사용자가 LMmaster를 처음 실행할 때 동의 버튼을 누른 시점부터 적용돼요. 동의한 시점은 PC에 안전하게 저장되며, 동의서 내용이 크게 달라질 경우에만 다시 안내드려요.

본 동의서는 한국 소비자 보호 법령과 사용자가 거주하는 국가의 강행 규정으로 보장된 권리를 어떤 경우에도 제한하지 않아요. 일부 조항이 그 법령과 충돌하면 해당 부분만 효력 범위 안에서 조정되며, 나머지 조항은 그대로 유효해요.

---

## 1. 개요

LMmaster는 사용자의 PC에서 로컬로 동작하는 AI 운영 허브예요. 이 동의서는 LMmaster를 사용할 때의 사용자 권리와 의무를 정리해 드려요.

본 동의서는 LMmaster 본체에 대한 약속이에요. LM Studio, Ollama, 모델 가중치, Python 의존성 같은 외부 구성 요소는 각 제공자의 라이선스가 별도로 적용돼요. (자세한 내용은 아래 "외부 모델·런타임 책임" 항목 참고)

---

## 2. 사용 권한

LMmaster는 다음 범위에서 자유롭게 사용할 수 있어요.

- 개인적·상업적 용도 모두 사용 가능해요.
- 코드 수정과 재배포는 함께 제공되는 LICENSE 파일의 조건을 따라요.
- LMmaster로 만든 결과물은 사용자 본인 소유예요.

다만 다음 용도로는 사용을 금지해요. 위반 시 사용자 본인이 모든 법적·도의적 책임을 지게 돼요.

- 본인 또는 다른 사람의 권리(저작권·상표권·초상권·명예권·프라이버시·영업비밀 등)를 침해하는 콘텐츠를 만들거나 배포하는 행위.
- 미성년자에 대한 성적·폭력적 묘사를 만들거나 보관·배포하는 행위. 한국의 청소년보호법, 아동·청소년의 성보호에 관한 법률, 그리고 사용자 거주국의 관련 법령 위반에 해당해요.
- 보이스피싱·사기·해킹·악성코드·스팸 등 범죄 또는 부정 경쟁 목적의 사용.
- 특정 개인이나 집단을 향한 차별·혐오·위협·괴롭힘 표현을 만들거나 자동 생성하는 행위.
- 의료·법률·금융처럼 면허나 자격이 필요한 영역에서 LMmaster의 출력만으로 전문가의 자문을 대체하는 행위.

---

## 3. 외부 통신

LMmaster는 사용자의 데이터를 외부로 보내지 않는 것을 기본 원칙으로 해요.

- 사용자가 입력한 프롬프트·문서·이미지는 모두 PC 안에서만 처리돼요.
- 새 버전이 있는지 확인하기 위해 6시간에 한 번 GitHub의 릴리즈 정보만 가져와요. 설정에서 끌 수 있어요.
- LM Studio·Ollama 같은 외부 런타임은 각 제공자의 정책이 따로 있어요. 그 정책은 LMmaster가 다시 보여드리지 않으니 직접 확인해 주세요.
- 익명 사용 통계는 사용자가 직접 켤 때만 한 곳의 자체 호스팅 수집기로 전송돼요.

---

## 4. 데이터 보관

- 워크스페이스 데이터(대화·설정·받은 모델)는 사용자 PC의 LMmaster 폴더에만 저장돼요.
- 익명 통계는 기본적으로 꺼져 있어요. 사용자가 직접 켜면 운영체제 종류, GPU 모델, VRAM 같은 PC 사양만 익명으로 전송돼요. 프롬프트, 모델 출력, 파일 내용은 어떤 경우에도 전송되지 않아요.

---

## 5. 책임 한계

- LMmaster는 "있는 그대로(AS IS)" 그리고 "이용 가능한 상태대로(AS AVAILABLE)" 제공돼요. 적용 가능한 법령이 허용하는 범위에서, 상품성·특정 목적 적합성·정확성·중단 없는 동작 같은 보증은 명시적이든 묵시적이든 제공하지 않아요.
- LMmaster 사용 중에 다음과 같은 일이 생겨도 LMmaster 측은 책임지지 않아요.
  - 사용자 PC의 데이터 손실, 워크스페이스 파일이나 모델 가중치의 손상.
  - AI 모델이 만든 답변이 부정확하거나 편향됐거나, 사실과 다르거나, 다른 사람의 권리를 침해한 경우.
  - LM Studio·Ollama·llama.cpp·Python 같은 외부 런타임의 장애나 정책 변경.
  - 카탈로그 갱신이나 자동 업데이트 확인이 잠시 실패하는 경우.
- 사용자가 외부에 공개·배포·상업 이용하는 모델 출력의 적법성과 윤리적 책임은 사용자 본인이 부담해요.
- LMmaster는 무상으로 배포되는 소프트웨어이기 때문에, 본 소프트웨어로 인한 직접·간접·결과적 손해에 대한 LMmaster 측의 총 책임 한도는 적용 가능한 법령이 허용하는 범위에서 사용자가 LMmaster 측에 실제로 지급한 금액(통상 0원)으로 한정돼요.
- 위 면책과 책임 한도는 LMmaster 측의 고의 또는 중대한 과실로 인한 손해, 그리고 한국 약관규제법·전자상거래법, 사용자 거주국의 강행 규정에 따라 면책이 허용되지 않는 책임에는 적용되지 않아요. 그 범위에서는 관련 법령에 따라요.

---

## 6. 동의서 변경 안내

- 표현이나 오타를 다듬는 작은 수정은 별도 안내 없이 반영돼요.
- 사용자가 알아둘 만한 내용 변경이 있으면 다시 동의를 요청드리고, 무엇이 바뀌었는지 함께 보여드려요.

---

## 7. 외부 모델·데이터셋·런타임 책임

LMmaster를 통해 사용자가 받거나 실행하는 외부 모델 가중치(Llama, Qwen, Gemma, EXAONE, HyperCLOVA X SEED 등)와 데이터셋, 런타임은 각 제공자의 라이선스와 사용 약관이 별도로 적용돼요.

- LMmaster는 외부 모델·데이터셋·런타임의 출력 정확성·안전성·합법성·라이선스 준수 여부를 보증하지 않아요.
- 모델의 사용·재배포·상업 이용 가능 여부는 각 모델의 모델 카드와 라이선스 문서를 사용자가 직접 확인해 주세요. 일부 모델은 자체 사용 정책을 따로 두고 있어요.
- 모델이 만든 콘텐츠가 다른 사람의 저작권·초상권·명예·프라이버시·영업비밀을 침해하지 않는지 확인하고 사용하는 것은 사용자 본인의 책임이에요.
- 모델이나 데이터셋을 다운로드할 때 발생하는 네트워크 비용, 디스크 사용량, 보안 위험에 대해서는 LMmaster 측이 책임지지 않아요.

---

## 8. 성인·민감 콘텐츠 정책

LMmaster의 카탈로그는 다음 원칙을 따라요.

- **미성년자 묘사 절대 금지** — 키워드 자동 검사와 사람 검토를 거쳐 등록되지 않아요. 발견 시 즉시 신고해 주세요.
- **명시적 성인 콘텐츠 표시** — 성인 표현이 가능한 모델·데이터셋은 명확하게 안내하고, 기본은 숨김 상태예요. 사용자가 카탈로그 상단의 토글로 직접 켤 때만 노출돼요.
- **검증된 라이선스만 등록** — Apache-2.0, MIT, OpenRAIL-M, CC-BY 같은 검증된 오픈 라이선스만 받아들여요.
- **사용자 책임** — 다운로드 후 사용은 사용자 PC 안에서만 일어나며, 거주국의 법(한국 청소년보호법 등)을 준수할 의무는 사용자 본인에게 있어요.

## 10. 일반 조항

- **준거법**: 본 동의서는 대한민국 법률에 따라 해석돼요. 다만 사용자 거주국의 강행 규정으로 보장된 권리는 그대로 유지돼요.
- **분쟁 해결**: 분쟁이 생기면 GitHub Issue 채널에서 먼저 협의해요. 협의가 어려우면 사용자 주소지나 거소지를 관할하는 대한민국 법원이 1심 전속관할 법원이 돼요.
- **분리 가능성**: 일부 조항이 무효이거나 집행할 수 없다고 판정돼도 나머지 조항은 영향 없이 유효해요.
- **양도**: 사용자는 본 동의서상의 권리·의무를 LMmaster 측의 사전 동의 없이 다른 사람에게 양도할 수 없어요.
- **수출 통제**: 사용자는 본 소프트웨어와 모델 가중치를 거주국의 수출 통제 법령(한국 대외무역법, 미국 EAR 등)을 위반하는 방식으로 반출하거나 재수출하지 않기로 약속해요.
