## 3. External Network Access

LMmaster's core principle is that your data stays on your PC.

- Prompts, documents, and images you provide are processed only locally.
- We check for new versions by fetching only the GitHub release index, once every 6 hours. You can disable this in Settings.
- External runtimes (LM Studio, Ollama, etc.) follow their own policies. We do not re-display them; please review them yourself.
- Anonymous usage telemetry is sent only when you explicitly opt in, to a single self-hosted endpoint.

---

## 4. Data Storage

- Workspace data (conversations, settings, downloaded models) is stored only in the LMmaster folder on your PC.
- Telemetry is off by default. When opted in, only anonymous PC stats (OS family, GPU model, VRAM) are sent. Prompts, model outputs, and file contents are never transmitted under any circumstance.

## 8. Adult & Sensitive Content Policy

The LMmaster catalog follows these principles:

- **No depictions of minors, ever** — Keyword scans and human review keep these out of the catalog. Report any you find immediately.
